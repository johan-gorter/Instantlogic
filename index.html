Instantlogic

Experimental framework for creating real-time, collaborative web applications.

Stop thinking about your web application as a data processor
Start thinking about your web application as a virtual reality

When using web applications built using instantlogic:

  1. Your changes are propagated real-time to everyone else's browser.
  2. You can see where other people are inside the application

Demonstration

Please take the time to watch this 3-minute demonstration.

Goal

The main goals for instantlogic are:

  1. Create real-time, collaborative web applications
  2. Make web application creation as non-technical as possible by using modeling techniques and business-rules
  3. Allow pain-free integration between modeling and programming

Roadmap

Instantlogic is still in its early stages of development. Just enough has been implemented to be able to give demonstrations. The next aim is to be able to create web application prototypes.
