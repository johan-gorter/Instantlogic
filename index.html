Instantlogic

Experimental framework for creating real-time, collaborative web applications.

Stop thinking about your web application as a data processor
Start thinking about your web application as a virtual reality


Tweet Follow @inlgc

When you are using a webapplication built with instantlogic, avatars mark the current location of every person using the application.

Every field you change is effective instantly. This means your data is always saved. There are also no 'ok' and 'cancel' buttons.

You can also make changes to the application itself. These changes are also effective immediately. The users (or testers) of your application even see your changes while you are editing.
